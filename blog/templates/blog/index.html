QRcode reader

{% load static %}

Thanks to Python, you can generate a QR code in 4 lines of script, while doing a QR code reader happens to be a bit more tricky. This reader is personally one of the most interesting projects I have done, given the versatility of elements to combine between mouse click events, image manipulation and using open CV for image analysing.
The script starts by maximizing a transparent window on which the x and y coordinates of the mouse clicks will be used as variables to generate an image, triggering the actual GUI of the QR code program, and the data contained will be displayed. An image-file.png will be created and you will need sudo privileges to run the file.
